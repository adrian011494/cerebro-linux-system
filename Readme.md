# cerebro-linux-system

> [Cerebro](https://cerebroapp.com) plugin to automate some Linux system features

## Usage

In Cerebro, type command that you need and `enter` to execute it. List of available system commands:

* `Restart`
* `Suspend`
* `Hibernate`
* `Shut Down`
